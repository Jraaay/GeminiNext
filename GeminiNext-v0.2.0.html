GeminiNext v0.2.0 Release Notes

Release Date / 发布日期: 2026-02-14

✨ What's New / 新功能

🎹 Custom Hotkey Recorder / 自定义快捷键录制器

Replaced the old preset hotkey dropdown with a fully custom hotkey recorder. You can now press any key combination to set your preferred shortcut for toggling the window. Hotkeys can also be completely disabled if not needed.

取代了旧版的预设快捷键下拉菜单，引入全新的自定义快捷键录制器。现在你可以按下任意组合键来设置你喜欢的窗口唤起快捷键。如果不需要，也可以完全清空（禁用）快捷键。

💬 New Chat Hotkey / 新建对话快捷键

Added a global hotkey to start a new conversation instantly. Default shortcut is Cmd + Shift + N, fully customizable or disableable via the same hotkey recorder.

新增全局快捷键，一键开启新对话。默认快捷键为 Cmd + Shift + N，同样支持通过快捷键录制器自定义或清空。

🚀 Launch Animation / 启动动画

Introduced a branded splash screen with a progress bar for the initial page load. Replaces the plain loading spinner with a polished launch experience.

引入品牌化闪屏页面，带有进度条指示首次加载状态。取代原本简陋的加载转圈，提供更精致的启动体验。

🧹 Clear Browsing Data / 清除浏览数据

Added a Clear Browsing Data option in Settings that removes cookies, cache, and local storage in one click. Includes a confirmation dialog warning that you'll need to re-login, and automatically reloads the WebView after clearing.

在设置中新增 清除浏览数据 选项，一键清除 Cookies、缓存和本地存储。操作前会弹出确认对话框，提醒用户清除后需重新登录，清除完成后自动刷新 WebView。

⬆️ Upgrade / 升级方式

Download the latest .dmg from Releases or update via app settings.

从 Releases 页面下载最新 .dmg，或在应用的设置内完成更新。
